Operadores aritméticos:

JavaScript tiene un amplio conjunto de operadores, entre los que se incluyen operadores aritméticos, lógicos, bit a bit y de asignación, así como algunos operadores varios.
Los operadores permiten hacer cálculos complejos y tomar decisiones lógicas de función de comparaciones y otros tipos de condiciones. Los operadores aritmeticos son adicion o suma (+), sustracción o restan (-), multiplicación (*), división (/), incremento (++), decremento (--).

Los tipos de datos principales (primitivos) de JavaScript son:

  • Cadena (string):
    Un valor de cadena está formado por una cadena de cero o más caracteres Unicode (letras, dígitos y signos de puntuación). El tipo de dato string se usa para representar texto en JavaScript. Numero

Eventos:

Un evento HTML puede ser algo que hace el navegador, o algo que hace un usuario. JavaScript puede “reaccionar” en estos eventos. Algunos eventos HTML comunes son: Onchange: Un elemento HTML ha sido cambiado. Onmouseover: El usuario hace clic en un elemento HTML. Onmouseout: El usuario aleja el mouse de un elemento HTML. Onload: El navegador ha terminado de cargar la página.

DOM (Document Object Model):

El Modelo de objetos de documentos (DOM) del W3C es una plataforma y una interfaz de lenguaje y una interfaz de lenguaje neutral que permite que los programas y los scripts accedan y actualicen dinámicamente el contenido, la estructura y el estilo de un documento. El DOM está definido y administrado por el W3C, por lo que los distintos navegadores simplemente aplican las especificaciones del World Wide Web Consortium, para dar soporte al Dom es sus aplicaciones. Con el DOM los programadores pueden construir documentos, navegar por su estructura, y añadir, modificar, o eliminar elementos y contenido. Se puede acceder a casi cualquier elemento que se encuentre en un documento HTML.

El HTML DOM es un modelo de objetos estándar y una interfaz de programación para HTML que define.

  • Los elementos HTML como objetos.
  • Las propiedades de todos los elementos HTML
  • Los métodos para acceder a todos los elementos HTML. En otras palabras: el HTML DOM es un estándar sobre como obtener, cambiar, agregar, o eliminar elementos HTML.
  • El DOM es una interfaz que implementa los navegadores para poder interactuar con los elementos HTML.

Por último, los objetos creados se vinculan en una estructura de datos en forma de árbol que muestra las relaciones entre objetos primarios y secundarios definidas en el HTML original.

El resultado de todo este proceso es de Document Object Model (DOM), que el navegador usa para todos los procesamientos de la página. Como muetra la grafica de arblo de abajo.

Referenciar con JS

Para poder manipular los elementos HTML con JS, primero debemos encontrarlos.
Esto lo podemos hacer de varias formas. Esto lo podemos hacer de varias formas:

Encontrar elementos por su id:

El elemento getElemetnById() encuentra al elemento que tiene el id especificado. Este método es uno de los más comunes en el HTML DOM, y se usa casi cada vez que desea manipular u obtener información de un elemento en su documento. Si no hay elementos en el id especificado devuelve null. Como ya sabemos, un id debe de ser único en un archivo HTML. Sin embargo, si existe más de un elemento con el id especificado, el método getElementById() devuelve el primer elemento en el código fuente.

Encontrar elementos por su nombre de clase.

El elemento getElementByClassName() devuelve una colección de todos los elementos en el socumento con el nombre de la clase especificado, como un objeto HTMLCollection (lista de elementos HTML). El objeto HTMLCollection representa una colección de nodos. Se puede acceder a los nodos por números de índice. El índice comienza en o.

Encontrar elementos por su nombre de etiqueta.

El método getElementByTagName() devuelve una colección de todos los elementos en el documento con el nombre de etiqueta especificado, como un objeto HTMLCollection. Encontrar elementos por su selector CSS. El método querySelector() devuelve el primer elemento que coincide con un selector CSS especificado en el documento. Si el selector coincide con una “id” en el documento que se usa en varias veces (hay que tener en cuenta que un “id” debe ser único dentro de una página y no debe usarse más de una vez), devuelve el primer elemento coincidente. El método querySelectorAll() devuelve todos los elementos del documento que coinciden con un selector “id” especificado, como un objeto NodeList estático. Cocumento.querySelectoAll(´selector CSS, o varios selectores separados por comas´).

ClassList

La propiedad ClassList mas el metodo add agrega uno o mas nombre de clase a un elemento. Si la clase especifica ya existe. la clase no se agregara. La propiedad es util par agretgar, eliminar, y alternar clases CSS en un elemento.
La propiedad ClassList es solo para lectura, sin embargo puede modificarla utilizando los metodos add(), remove() y toggle().

La propiedad ClassList mas el metodo remove elimina uno o mas nombre de un elemento. Eliminar una clase que no existe, NO arroja un error.

classlist.toggle()

La propiedad classList más el método toggle alterna entre un nombre de clase para un ejemplo. El primer parámetro elimina la clase especificada de un elemento y devuelve el falso.

Si la clase no existe, se agrega el elemento y el valor booleano que obliga a agregar o quitar la clase, independientemente de si ya existia o no.

Se podían defnir reglas d eestilos para computadoras, impresoras, dispositivos de mano, televisiones, etc.

@media CSS3

Las medias queries son un módulo de CSS3 que adapta la representación del contenido a las características de un dispositivo.

Se convirtio en un estándar recomendado por la W3C en junio de 2012 y es un principio basico del diseño web adaptativo.

CSS3 introdujo media queries que extendieron la idea de los tipos de media de CSS2: en lugar de buscar un tipo de dispositivo, analizan la capacidad del dispositivo.

Las media queries se pueden utiliazr para verificar muchas cosas, como: ancho y alto, orientación del dispositivo (horizontal o vertical) y la resolucion.

El uso de media queries es una técnica popular para entregar una hoja de estilo personalizada para computadoras de escritorio, computadoras portátiles, tabletas y telefonos móviles (como los teléfonos !Phone y Android).
